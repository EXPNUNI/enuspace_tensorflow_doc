# void enuSetModifyProject\(bool bFlag\)

void enuSetModifyProject\(bool bFlag\)

#### Parameters

* bool bFlag

Project의 수정 모드를 설정합니다.

#### Return Value

Type : NONE

#### Remarks

프로젝트의 변경에 대하여 모드를 설정합니다. 

#### Examples

```cpp
enuSetModifyProject(true);
```
