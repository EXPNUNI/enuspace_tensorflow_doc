# void enuSetProjectName\(wchar\_t\* projectname\)

void enuSetProjectName\(wchar\_t\* projectname\)

#### Parameters

* wchar\_t\* projectname

#### Return Value

Type : NONE

#### Remarks

프로젝트의 이름을 설정합니다.
