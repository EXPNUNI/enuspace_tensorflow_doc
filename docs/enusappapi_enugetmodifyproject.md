# bool enuGetModifyProject\(\)

bool enuGetModifyProject\(\)

#### Parameters

* NONE

#### Return Value

Type : bool

PROJECT의 변경유무를 반환합니다.

#### Remarks

PROJECT의 변경유무를 반환합니다.

Project에 파일추가되거나 제거되었을 경우 변경유무를 반환합니다.

#### Examples

```cpp
bool bModify = enuGetModifyProject();
```
