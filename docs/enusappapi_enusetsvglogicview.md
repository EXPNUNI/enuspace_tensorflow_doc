# bool enuSetSvgLogicView\(HVIEW pENUView, wchar\_t\* pStrFileName\)

bool enuSetSvgLogicView\(HVIEW pENUView, wchar\_t\* pStrFileName\)

#### Parameters

* HVIEW pENUView

뷰의 핸들을 입력합니다.

* wchar\_t\* pStrFileName

LOGIC영역의 SVG 파일이름을 입력합니다.

#### Return Value

Type : bool

정상적으로 뷰에 SVG핸들이 적용되었는지를 반환합니다.

#### Remarks

뷰와 SVG핸들을 이용하여 디스플레이를 수행합니다.
