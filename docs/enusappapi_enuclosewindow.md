# void enuCloseWindow\(wchar\_t\* strWindow\)

void enuCloseWindow\(wchar\_t\* strWindow\)

#### Parameters

* wchar\_t\* strWindow

런타임뷰의 윈도우 닫고자하는 윈도우 이름을 입력합니다. 

#### Return Value

Type : NONE

#### Remarks

런타임뷰의 열려있는 윈도우를 닫기를 수행합니다.

#### Examples

```cpp
enuCloseWindow(L"window");
```
